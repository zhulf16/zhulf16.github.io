ARRAY

APA ITU ARRAY?

Array merupakan sebuah variabel yang menyimpan lebih dari satu buah data yang memiliki tipe data yang beragam, jadi dapat dikatakan bahwa array merupakan kumpulan dari data-data tunggal yang dijadikan dalam 1 variabel array yang alamat memorinya berbeda yang selanjutnya disebut elemen-elemen array yang bisa diakses berdasarkan indeks.

Array dalam kehidupan sehari hari dapat ditemukan pada penyusunan buku di perpusktakaan, dimaana dalam penyusunan dilakukan pada bebereapa kategori, contohnya perpuskataakan di Kampus, maka buku - buku yang ada didalam perpusktakaan akan dibedakan berdasarkan Fakultas, jurusan, mata kuliah, tingkatan (Pemula sampai Ahli), dan berbagai pengelompokan lainnya.



CARA MEMBUAT ARRAY DALAM BAHASA JAVASCRIPT

Pada Javascript Array dapat kita buat dengan menggunakan tanda kurung siku ( [ ... ] ) Contoh :

var zulfi = [ ];
Maka variabel zulfi nantinya akan berisi sebuah array kosong

Kita dapat mengis data kedalam array yang masih kosong tadi, yang setiap datanya dipisah menggunakan tanda ( , ) Seperti pada contoh berikut ini : teplate untuk contoh

var zulfi = ["Bogor", "Tasik", "Februari"];

ini artinya, didalam array zulfi sudah berisi Bogor, Tasik, Februari yang perlu diketahui dalam array pada bahasa Javascript ini bahwa indeks atau perhitungan data didalam array dimulai dari 0, jadi didalam array zulfi berisikan 2 buah indeks yang mana Bogor = 0, Tasik = 1, dan Februari = 2.

Javascript adalah bahasa pemrograman yang bersifat Dynamic Typing, artinya kita dapat mencampur dan menyimpan array dengan beberapa tipe data, Contohnya seperti ini :

var zulfi = ["Bogor", "Tasik", True, 16, 0.5];
Contoh sederhana dari array pada Javascript yang akan kita tampilkan keadalam html
(script)
zulfi = ["Bogor", "Tasik", "Februari"];

const ctharray = document.getElementById('arrayjs'); ctharray.innerHTML = zulfi;
(/script)
maka yang akan tampil pada html adalah seperti berikut dan ini ditulis dalam bahasa Javascript
