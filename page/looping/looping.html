APA ITU LOOPING

Looping atau perulangan adalah suatu serangkain perintah untuk mengulangi suatu kondisi atau program tertentu.

Misalnya jika kita ingin menampikan sepuluh kata Hello world pada console.log di bahasa memprograman javascript kita dapat menuliskannya seperti ini

console.log('Helo World')
console.log('Helo World')
console.log('Helo World')
console.log('Helo World')
console.log('Helo World')
console.log('Helo World')
console.log('Helo World')
console.log('Helo World')
console.log('Helo World')
console.log('Helo World')

Jika kita melihat console maka akan ada 10 kali pengulangan kata Hello World namun, cara tersebut tidaklah efektif, contohnya jika kita ingin menulisnya sebanyak 1000 kali, tentunya akan sangat menyulitkan.

MENGENAL PERULANGAN PADA JAVASCRIPT

Kali ini kita akan membahasa 3 jenis pengulangan :

1. Perulangan FOR
2. Perulangan WHILE
3. Perulangan DO/WHILE

PERULANGAN MENGGUNAKAN FOR

Perulangan FOR merupakan perulangan yang termasuk dalam counted loop karena sudah jelas berapa kali FOR harus melakukan looping, Contoh sintaks untuk melakukan looping menggunakan FOR :

for (var i = 0; i < 10; i++){
console.log('Hello World');
}

Maka akan tertulis kata Hello world sebanyak 10 kali pada konsole log, tentunya itu akan memudahkan kita ketika kita akan melakukan perulangan yang banyak.


Mari kita bedah satu persatu :

Var i = 0 itu artinya perulangan yang kita lakukan akan dimulai dari 0
i < 10 Perulangan yang kita lakukan akan dilakukan sebanyak 10 kali
i++ Itu artinya setiap perulangan akan ditambah 1 ( Increament )

PERULANGAN MENGGUNAKAN WHILE

Contoh sintaksnya adalah sebagai berikut :

var nilaiAwal = 0;

while (nilaiAwal < 10) {
console.log('Hello World');

nilaiAwal++;
}

Sama seperti for, perintah tersebut akan mengembalikan nilai Hello World sebanyak 10 kali kedalam konsole log.



PERULANGAN MENGGUNAKAN DO/WHILE

Perlungan dengan DO/WHILE akan melakukan perulangan sebanyak 1 kali terlebih dahulu, lalu mengecek kondisi yang ada didalam kurung WHILE.
Contoh sintaksnya seperti berikut :

nilaiAwal = 1

do {
console.log ('Hello World')
nilaiAwal++

} while (nilaiAwal <= 10)

Mungkin akan sedikit menyulitkan menggunakan perulangan DO/WHILE bagi yang tidak terbiasa hehe :)

Oke langsung saja kita akan melakukan perulangan menggunakan FOR yang ditulis dengan bahasa javascript dan hasilnya akan muncul dibawah ini
