APA ITU FUNCTION

Function atau fungsi adalah sub-program yang bisa digunakan kembali baik didalam program itu sendiri maupun pada program lain.
Fungi didalam Javascript adalah sebuah objek, karena memiliki properti dan juga method

CARA MEMBUAT FUNGSI PADA JAVASCRIPT

Cara yang paling sering digunakan pada Javascript dalam membuat suatu fungsi biasanya dilakukan dengan cara berikut :

Function = namaFungsi( ){
console.log ('Hello world!'); }
Artinya kita mencetak kalimat Hello world kedalam konsol pada browser

MEMBUAT FUNGSI DENGAN PARAMETER

Kali ini kita akan mecoba membuat fungsi menggunakan parameter atau variabel untuk menyimpan suatu nilai yang nantinya akan diproses oleh Function

Function kali (a, b){
hasilKali = a * b;
console.log('Hasil kali a*b = ' + hasilKali);
}

Artinya kita telah membuat fungsi yang memiliki parameter a dan b. Yang mana nilai dari masing-masing parameter masih kosong. lalu kita akan mengisi nilai parameter tersebut

kali(3, 2);

Sekarang parameter a memiliki nilai 3 dan parameter b memiliki nilai 2, lalu karna kita telah memasukan fungsi aritmatika ( * ) maka nilai dari masing-masing parameter akan kita kali-kan. ketika kita melihati lagi console.log maka akan mengembalikan nilai

function bagi(a,b){

// memanggil fungsi
var nilai1 = 50;

//output ditampikan pada console
console.log(hasilPembagian);

Artinya kita memiliki fungsi yang bernama bagi yang memiliki 2 parameter yaitu a dan b dan masing-masing parameter memiliki nilai 50 dan 10 lalu kita memasukan operasi aritmatika sederhana yaitu ( / ) bagi kedalam fungsi tersebut.
Selanjutnya kita memiliki sebuah variable yang bernama hasilPembagian yang memiliki nilai dari fungsi bagi diatas, dan kita akan mengeluarkan atau menampilkan hasil dari fungsi tersebut kedalam console.log
