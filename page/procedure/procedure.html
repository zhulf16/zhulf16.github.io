APA ITU PROCEDURE?

Pengertian Prosedur, Fungsi, dan Method itu sama.

Prosedur adalah sebutan untuk fungsi yang tidak mengembalikan nilai.
Fungsi adalah sebutan untuk fungsi yang mengembalikan nilai.
Method adalah fungsi yang berada di dalam Class. Sebutan ini, biasanya digunakan pada OOP.

Berikut ini contoh sintaks Prosedur :

const procedure = document.getElementById('procedurejs')

function ini ( a ) {
var a = ' ';
return a;
}

var a = 'DCI';
var hasil = a += " ";

// memanggil fungsi
procedure.innerHTML = hasil;
Maka akan tampil pada HTML seperti ini

Itulah pembahasan singkat kita mengenai materi procedure,
